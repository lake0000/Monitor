# Roadmap

## V1 已完成

- WinForms 桌面界面。
- 系统托盘常驻。
- 文件变化监听。
- SQLite 本地记录。
- 增长和减少变化展示。
- 显示阈值可调。
- 目录聚合。
- 点击条目查看详情和溯源树。
- 忽略列表。
- 暂停、恢复、手动扫描。
- CSV 导出。

## 后续可考虑

- 更完整的设置页。
- 忽略列表管理界面。
- 更细的时间范围过滤。
- 便携版打包。
- 开机自启选项。
- 更丰富的导出格式。

## 不计划加入 V1 的能力

- 自动清理。
- 一键删除。
- 系统优化。
- 注册表清理。
- 文件正文分析。
- 进程级 I/O 归因。
